# Valhalla Fitness

Aplicación de fitness gamificada con estética heavy metal, diseñada de forma personalizada para un cliente baterista.

## Objetivo del proyecto

El objetivo de esta app es transformar una rutina de ejercicio en una experiencia más motivadora, visual y cercana a la identidad del usuario.

En lugar de utilizar una plantilla genérica de fitness, el proyecto adapta el tono, la estética y la motivación a una persona concreta, conectando entrenamiento, música y gamificación.

## Características principales

- App de fitness personalizada.
- Estética inspirada en el heavy metal y el imaginario vikingo.
- Enfoque gamificado para reforzar la constancia.
- Diseño adaptado a un usuario baterista.
- Proyecto ligero orientado a navegador.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Diseño de aplicaciones personalizadas para un usuario concreto.
- Aplicación de gamificación a rutinas de salud y ejercicio.
- Creación de una identidad visual coherente con el perfil del cliente.
- Organización de interfaces sencillas orientadas a hábitos.
- Desarrollo web con HTML, CSS y JavaScript.
