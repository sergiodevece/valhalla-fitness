# Valhalla Fitness

**Valhalla Fitness** es una aplicación de fitness gamificada y personalizada, diseñada con estética heavy metal/vikinga para convertir el entrenamiento semanal en una experiencia más épica, visual y motivadora.

El proyecto está orientado a una persona concreta, **Mansanito**, con una identidad visual propia y una lógica de seguimiento pensada para que la rutina no parezca una tabla gris de gimnasio pegada en la nevera.

## Concepto

La idea principal es transformar una rutina de ejercicio en una pequeña aventura personal.

En lugar de presentar los entrenamientos como una lista fría de tareas, Valhalla Fitness los convierte en un sistema visual de progreso, semanas, recompensas y acciones completadas. La estética mezcla heavy metal, runas, fuego, imaginería vikinga y una interfaz oscura con personalidad propia.

## Funcionalidades principales

- App de fitness personalizada para un usuario concreto.
- Estética inspirada en heavy metal, runas y universo vikingo.
- Selector de semanas para organizar el progreso.
- Sistema de pestañas para navegar por secciones de la app.
- Vista del día actual.
- Registro visual de ejercicios completados.
- Botones de entrenamiento con estilo de pads/batería.
- Sección de caminata o actividad cardiovascular.
- Consejos y notas de apoyo.
- Sección de rutina con ejercicios y progresión semanal.
- Sistema de recompensas visuales.
- Diseño responsive pensado para uso en móvil.

## Identidad visual

La app utiliza una estética oscura y dramática, con:

- fondos negros y rojos profundos;
- tipografías decorativas y metaleras;
- detalles tipo runas;
- líneas de fuego;
- botones circulares inspirados en pads de batería;
- una interfaz pensada para reforzar motivación y carácter.

Esta dirección visual conecta especialmente bien con el perfil del usuario final: un cliente baterista, ligado a la música y a una estética más cañera que la de una app fitness genérica.

## Estructura del proyecto

El proyecto está planteado como una aplicación web ligera, orientada a navegador y pensada para poder usarse de forma sencilla desde el móvil.

```txt
valhalla-fitness/
├── README.md
└── aplicación HTML principal
```

## Cómo usarlo

1. Abrir la aplicación en el navegador.
2. Seleccionar la semana correspondiente.
3. Revisar la rutina o los ejercicios del día.
4. Marcar los ejercicios completados.
5. Consultar el progreso y las recompensas visuales.
6. Mantener la constancia sin perder el tono épico del proyecto.
